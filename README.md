# 🧠 News Popularity Prediction using NLP

## 📌 Overview
This project predicts how attention-grabbing a news article is using NLP and Transformer models.

## 🚀 Features
- Emotion Detection
- Urgency Score
- Lexical Diversity
- Transformer Embeddings
- Explainable Output

## 🧠 Model
- DistilBERT for embeddings
- Rule-based scoring system

## 💻 Tech Stack
- Python
- Streamlit
- Transformers
- TextBlob
